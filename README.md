Organización de Planilla de Alumnos

Este programa tiene como objetivo ayudar a un profesor a organizar la información de sus alumnos de manera eficiente. 
Permite cargar nombres, apellidos y notas de los estudiantes, generando automáticamente información adicional y visualizaciones útiles.
Librerías necesarias:
pandas
csv
matplotlib
Al iniciar el programa, se solicita al usuario que ingrese los apellidos y nombres de los alumnos, junto con dos notas para cada uno.
El programa realiza los cálculos necesarios:
-Promedio de cada alumno.
-Condición final de acuerdo a las notas ingresadas.
-Se genera un archivo Alumnos.csv con toda la información organizada.
-Se genera un gráfico de barras titulado "Promedio de los alumnos", donde se visualizan los promedios de cada estudiante.

Este proyecto es uno de mis primeros desarrollos en Python y combina el manejo de archivos CSV, lógica de negocio y visualización de datos.
